<objective>
Scope the quality review to a selected cohort/subcohort and honor that cohort's persisted quality-parameter selection (QUAL-020). The QualityPage gains a cohort selector: when a saved cohort is chosen, the reviewed case set is restricted to that cohort's cases, and the per-case anomaly checks run only the parameters that cohort selected (40-02's qualityParams), falling back to ALL checks when the cohort has no selection or none is chosen.

Purpose: Today QualityPage reviews the global case set with a fixed check set. QUAL-020 lets a reviewer focus on one cohort, and QUAL-021's persisted selection (40-02) now drives which checks apply to that cohort.

Output: A cohort scope control on QualityPage, cohort-filtered cases + counts, qualityParams threaded into the anomaly computation with an all-checks fallback, i18n, and a behavior test. Reuses Phase 39 threshold accessors (already in QualityCaseDetail) — no threshold changes.
</objective>

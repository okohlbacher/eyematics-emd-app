<objective>
ANL-010: When comparing cohorts, every compare plot — including the interval histogram (the user-flagged gap) — must clearly label which cohort each series represents, by color + name, consistent with the existing palette.

The Visus and CRT trajectory panels already overlay per-cohort series with named legend chips (OutcomesPanel cross-mode, `name={cohortName (N=...)}`, color from COHORT_PALETTES). The interval histogram and responder view do NOT — in cross mode they currently receive only the primary cohort's cases (`cohort.cases`) and render a single unlabeled series. This plan closes that gap for the interval histogram (primary flagged gap) and applies the same labeled-legend treatment to the responder view so all four metric tabs are consistent in compare mode.

Purpose: A user comparing cohorts can read every plot and know which cohort is which.
Output: Labeled, palette-colored, legend-bearing interval histogram (and responder view) in cross-cohort mode; single-cohort rendering unchanged.
</objective>

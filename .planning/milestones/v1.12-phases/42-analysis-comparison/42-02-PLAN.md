<objective>
ANL-011: The Aggregated tab supports between-cohort comparison — at minimum diagnosis distribution and an age-vs-Visus comparison — rendered per-cohort using the same COHORT_PALETTES palette as the Trajectories compare plots.

This plan reads the cross-cohort selection (the same `?cohorts=` deep-link OutcomesView consumes), resolves each cohort's cases, and renders comparable per-cohort diagnosis distribution and age-vs-Visus views. Scope is comparison RENDERING of existing aggregate data — no new metrics (per CONTEXT: deferred-ideas excludes new aggregate metrics).

Purpose: A user comparing cohorts can compare their diagnosis mix and age-vs-Visus side by side, not just trajectories.
Output: Cross-cohort comparison rendering in the Aggregated tab; single-cohort tab unchanged.
</objective>

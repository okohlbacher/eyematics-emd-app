## Dark mode

**You wrote:** (a) "When in System design (light) the cycle is on System but does not skip the Light mode (which it should)." · (b) "Single tables and plots are still not changing to dark mode."

**What changed:**
- **(b) Remaining figures/tables now restyle in dark mode** — the Dokumentationsqualität benchmarking chart and its tables, the per-centre detail chart, plausibility/score tables, the filter bar, and the other charts that were still light now follow the theme.
- **(a) Theme toggle.** We looked hard at "skip the redundant step": skipping it actually *strands* a mode — once you leave it, you can no longer cycle back to "System" under a dark OS. So we kept the full **Light → Dark → System** cycle, where every mode is reachable. The trade-off is that one click per OS setting is a no-visible-change step (e.g. System→Light when your OS is already light) — but the icon/label still change, and you can always reach all three modes. If you'd prefer a direct Light/Dark/System selector (three buttons, no cycling) instead, that's a quick change — just say so.

**How to verify:** set the OS (or pick a mode) to dark → every figure and table on Dokumentationsqualität and the case view renders dark; the toggle reaches Light, Dark and System.

**Status:** ✅ Fixed (full dark-mode coverage); ⚪ theme-cycle kept as a reachable 3-way cycle (see note).

## Dokumentationsqualität — "Vollzähligkeit" time-range filter (new)

**You wrote:** "The Vollzähligkeit metric seems false when filtering the time range. The absolute number of patients remains at 1775 and is not restricted to the range."

**What changed:** the displayed patient count now **restricts to the selected period** (the active patients in that range). The completeness **percentage** keeps the full registered population as its denominator, so narrowing the range **lowers** the reading (it reads as "share of the registered population documented in this period") rather than collapsing to ~100%. The "1775" you saw was the denominator (full registered total); the number that now follows your filter is the active-patient count.

**How to verify:** Dokumentationsqualität → Zeitraum → "Letztes Jahr": the count drops (e.g. 245 → 71) and Vollzähligkeit moves accordingly (e.g. 14% → ~4%), instead of staying pinned.

**Status:** ✅ Fixed.

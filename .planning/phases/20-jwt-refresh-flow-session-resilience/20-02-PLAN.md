| T-20-13 | Tampering (algorithm confusion via future direct import) | All TS/TSX files | mitigate | ESLint `no-restricted-imports` blocks compile-time; jwtUtil.ts + keycloakJwt.ts are the SOLE allowed importers; rule verified by throwaway probe in Task 1 |
| T-20-14 | Repudiation (post-credential-change refresh token reuse) | 4 mutation endpoints | mitigate | tokenVersion bump on every mutation (D-18); refresh endpoint enforces `payload.ver === user.tokenVersion`; tested in credentialMutationInvalidation.test.ts (5 mutation tests + 1 regression) |

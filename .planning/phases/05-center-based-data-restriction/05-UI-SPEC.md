# Phase 5 — UI Design Contract

> Visual and interaction contract for Phase 5: Center-Based Data Restriction.
> Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Phase UI Surface

Phase 5 is primarily a server-side enforcement phase. The UI surface is narrow but precise:

| Surface | What Changes |
|---------|-------------|
| CohortBuilder center filter | Shows only centers returned by server — no non-permitted centers visible (D-15, D-16, D-17) |
| DataContext loading state | Same `loading` spinner during `/api/fhir/bundles` fetch — no visual change, pattern reuse |
| DataContext error state | New error banner if `/api/fhir/bundles` returns 403 or network failure |
| Data mutation error | 403 response on write (case outside user's centers) surfaces via existing `mutationError` banner |
| Admin bypass | No UI change — admin sees all centers as before; bypass is transparent to the user |

No new pages. No new nav items. No destructive UI flows introduced in this phase.

## Interaction Contracts

### Center Filter — No Interaction Change

The center filter checkboxes require no interaction change. The set of rendered centers shrinks to the user's permitted set because `extractCenters()` now derives from server-filtered FHIR bundles. The executor must NOT add any visible permission badge or lock icon — the restriction is silent by design (D-15, D-16).

### 403 on FHIR Load — No Retry by Default

A 403 on `/api/fhir/bundles` means the user has no permitted centers or their JWT is invalid. The error state must:
1. Show the 403 heading + body (bilingual per i18n key)
2. NOT show a retry button (retrying will produce the same 403)
3. Show a "Contact administrator" paragraph only

A network error (5xx, timeout) shows the Retry button.

### 403 on Data Mutation — Auto-Dismiss Toast

Reuse the existing `mutationError` auto-dismiss pattern (4 s). Map HTTP 403 responses from `/api/data/*` writes to the `mutationForbiddenCase` or `mutationForbiddenSearch` i18n key. Do not block the UI; the operation simply fails silently after the toast.
